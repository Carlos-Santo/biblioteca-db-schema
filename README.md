# 📚 Biblioteca DB Schema

Este repositório contém o schema do banco de dados do sistema **Biblioteca**.  
Inclui a definição de tabelas, relacionamentos, constraints e demais estruturas necessárias para a criação do banco.

---

## 📌 Conteúdo

- Estrutura das tabelas
- Relacionamentos entre entidades
- Constraints (PRIMARY KEY, FOREIGN KEY, etc.)
- Script SQL para criação do banco
